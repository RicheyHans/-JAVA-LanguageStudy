# JAVA Basic Language
## Chapter 01. About JAVA

### 1.1 프로그래밍 언어
* 프로그래밍 언어 구분 <br>
    * 고급 언어 : 프로그래밍 언어 중, 사람이 상대적으로 이해하기 쉬운 언어. 기계어(0101010000...) 번역을 위한 컴파일(Compile) 과정 필요
    * 저급 언어 : 프로그래밍 언어 중, 기계어에 가까워 사람이 학습 / 이해하기 어려운 언어.


> 프로그램 : 프로그래밍 언어로 작성된 소스(source)를 기계어로 번역한 것.
